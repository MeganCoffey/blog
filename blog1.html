INCYDK: How the Web Works in Acronyms

  • ISP - Internet Service Provider
  • CPU - Central Processing Unit
  • RAM - Random Access Memory
  • TCP - Transmission Control Protocol
    • Connects the application to the port
  • OS - Operating Systems Software built on top of code
  • GUI - Graphical User Interface)
  • I/O - input/output (file systems and networking data)
  • HTTP - Hyper Text Transfer Protocol. Set of headers that web applications expect to receive and respond to
  • HTML - Hypertext Markup Language
    • Used to markup content. “tags” and “elements” that have meaning
  • CSS - Cascading Style Sheets Presentation Language
  • JS – JavaScript Internal presentation of HTML. Only in browser but not on server.
  • AJAX - Asynchronous JavaScript and XML Enables developer to make HTTP requests without having to reload the page or navigate to a new one
  • CDN - Content Delivery Network Sharing of files via hosting
  • URL - Uniform Resource Locator web address
